# Windows Server & Active Directory Home Lab

A hands-on Home Lab project focused on building and managing an enterprise Windows Server environment using Oracle VirtualBox. This project demonstrates practical experience with Active Directory, DNS, domain management, file sharing, permissions, and enterprise administration.

## 🖥️ Lab Environment

* **Virtualization:** Oracle VirtualBox
* **Server OS:** Windows Server 2022
* **Client OS:** Windows 10 Pro
* **Domain Name:** `crazyhub.local`

### Virtual Machines

| Machine | Role                            |  RAM |
| ------- | ------------------------------- | ---: |
| DC001   | Domain Controller (AD DS & DNS) | 2 GB |
| PC001   | Domain Client                   | 3 GB |
